# 用户算法使用说明

## 适用范围

用户算法用于把你的竞彩足球判断方法转换成平台可校验、可回测、可执行的 DSL。平台不会执行用户上传的 Python、JavaScript、SQL 或 Shell 代码，也不会把回测结果解释成未来收益承诺。

当前生产执行仅支持竞彩足球用户算法，目标玩法为：

- `spf`：胜平负
- `rqspf`：让球胜平负

## 设计算法

推荐从“我的算法”里的平台模板开始，再用 AI 助手或手工调整参数：

- 进球率预测法：看主队近六场进球、客队近六场失球、赔率和伤停。
- 最近六场预测法：看近六场胜场、净胜球和主客状态差。
- 赔率热度过滤法：过滤过低赔率热门，避免只追市场最热结果。
- 主客强弱差法：看排名差、状态差和让球方向。
- 伤停修正法：在赔率底座上修正伤停影响。
- 赔率变化法：看欧赔变化与基本面是否一致。

一个算法通常包含四部分：

- `filters`：先过滤不适合下注的场次，例如主胜赔率低于 1.35 时跳过。
- `score`：对字段加权评分，例如进球率、失球率、排名差、伤停优势。
- `decision`：决定预测目标、最低信心门槛、是否允许跳过。
- `targets`：指定回测和执行的玩法，例如只测 `spf` 或同时测 `spf`、`rqspf`。

设计时优先保持简单。先让一个清晰假设跑通，再用回测结果调整过滤条件和信心门槛。

## 理解回测

回测是在本地已开奖竞彩足球历史数据上重放算法。核心指标含义如下：

- 样本数：进入回测的已开奖场次数。
- 有效样本数：至少有一个目标玩法能判断命中或未命中的样本数。
- 字段完整率：赔率、近期战绩、积分排名、伤停、欧赔快照等字段的平均完整程度。
- 命中率：预测结果与实际赛果一致的比例。
- 跳过率：算法因过滤条件或信心门槛未出手的比例。
- 连红/连黑：按时间顺序统计最长连续命中和最长连续未命中。
- 最大回撤：按每场 1 注模拟时，从历史最高净值到后续最低净值的最大下跌。
- 跳过原因统计：区分过滤条件未满足、低于最低信心阈值、未生成预测等原因。
- 赔率区间表现：观察算法在低赔率、中赔率、高赔率区间的命中率。
- 可信度：综合有效样本数、实际出手数、跳过率、字段完整率和样本偏差得出的参考等级。
- 样本偏差：提示样本是否集中在单一联赛、单一玩法、短时间窗口或极端赔率区间。

回测支持筛选：

- 时间范围
- 最近 N 场
- 联赛
- SPF / RQSPF 单独回测

筛选越细，样本越少。样本少于 20 场时不要用命中率判断算法优劣，优先补齐历史赛果和赔率。

## 数据质量

回测质量取决于本地历史数据。字段缺失会直接影响算法评分：

- 缺近期战绩会影响进球率、最近六场、主客状态差。
- 缺伤停会影响伤停修正。
- 缺欧赔快照会影响赔率变化法。
- 缺 RQSPF 赔率或让球数会影响让球胜平负回测。

如果字段完整率偏低，应先补历史数据，再调整算法。

管理员可以在后台“竞彩足球数据健康”面板查看本地历史覆盖率，并触发历史补齐任务。平台也会按配置定时补齐最近一段已开奖赛事，补齐结果会记录为任务，包含日期范围、成功场次数、详情数和失败原因。

## 调参建议

回测后可以用一键调参或“AI 调参生成新版本”，旧版本不会被覆盖：

- 更保守：提高最低信心门槛，减少低质量出手。
- 更激进：降低最低信心门槛，增加出手机会。
- 降低跳过：放宽过滤条件，减少空场。
- 提高赔率价值：过滤过热低赔率，增加赔率一致性权重。

AI 调参会把当前算法定义和最近回测摘要一起发给你配置的 OpenAI 兼容接口。返回的新 DSL 会先经过平台校验，再保存为新版本；如果校验失败，会以草稿版本保留，方便继续修改。

每次调参都会创建新版本。预测记录会保存算法 key、版本和快照，便于以后追溯当时使用的算法定义。

版本列表会显示每个版本最近一次回测摘要，并对比上一版本的命中率变化。版本对比只能说明同一批历史样本或近似筛选条件下的相对表现，不应把单次提升直接理解为长期收益提升。

## 生产执行

用户算法绑定到竞彩足球预测方案后，平台会记录每次执行日志：

- 使用的算法版本、预测方案和批次。
- 本批次比赛数、实际生成预测数、跳过数和耗时。
- 用户算法失败时的处理策略和是否发生降级。
- 错误摘要，便于排查 DSL、字段缺失或运行时数据问题。

预测方案可以选择三种失败处理策略：

- 失败即停止：用户算法报错时本次预测失败，适合严格验证期。
- 内置基线：用户算法报错时临时降级到平台内置赔率基线算法，并记录降级。
- 跳过：用户算法报错时本批次不出手，只留下失败/跳过记录。

正式运营建议先用“失败即停止”暴露问题；稳定后再按业务需要选择“内置基线”或“跳过”。

## 风险边界

回测只说明算法在已有历史样本中的表现，不代表未来结果。

请特别注意：

- 足球比赛受阵容、临场战术、赛程、天气、红牌、赔率变化等多因素影响。
- 高命中率可能来自样本过少、赔率过低或过滤过多。
- 模拟收益未计入真实交易限制、投注限额、盘口变化、撤单和心理执行偏差。
- 任何算法结果都不能作为收益承诺或投注建议。

合理用法是把回测作为算法筛查工具，而不是把它当成盈利证明。
