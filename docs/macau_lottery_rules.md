# 新澳门六合彩 中奖规则说明

> 来源：macaujc.com  
> 所有投注派彩彩金皆含本金

---

## 基本概念

### 开奖号码结构

每期开奖共 **7 个号码**：
- **正码**：前 6 个号码（正一码 ~ 正六码，按落球顺序排列）
- **特码**：最后 1 个号码

### 号码范围

1 ~ 49 共 49 个号码球

---

## 特码玩法

### 特码大小

| 类型 | 条件 |
|------|------|
| 特小 | 01 ~ 24 |
| 特大 | 25 ~ 48 |
| 和局 | 49 |

### 特码单双

| 类型 | 条件 |
|------|------|
| 特单 | 奇数（01, 11, 35, 47...） |
| 特双 | 偶数（18, 20, 34, 42...） |
| 和局 | 49 |

### 特码合数单双

合数 = 十位数 + 个位数

| 类型 | 示例 |
|------|------|
| 合双 | 02(0+2=2), 11(1+1=2), 33(3+3=6), 44(4+4=8) |
| 合单 | 01(0+1=1), 14(1+4=5), 36(3+6=9), 47(4+7=11) |
| 和局 | 49 |

---

## 总和玩法

### 总和大小

所有 7 个开奖号码之和：

| 类型 | 条件 |
|------|------|
| 总和大 | ≥ 175 |
| 总和小 | ≤ 174 |

### 总和单双

| 类型 | 条件 |
|------|------|
| 总和单 | 总和为奇数 |
| 总和双 | 总和为偶数 |

## 组合玩法

### 三全中

- 选 3 个号码为一组（至少 3 个号）
- 3 个号码都在正码（1~6号，不含特码）中 = 中奖
- 其他情形 = 不中奖

### 三中二

- 选 3 个号码为一组（至少 3 个号）
- 中二：2 个号码在正码中 = 中奖
- 中三：3 个号码都在正码中 = 中奖（更高赔率）
- 只中 1 个或没中 = 不中奖

### 二全中

- 选 2 个号码为一组（至少 2 个号）
- 2 个号码都在正码中 = 中奖
- 一个在正码一个在特码 = **不中奖**

### 二中特

- 选 2 个号码为一组
- 中二：2 个号码都在正码中 = 中奖
- 中特：一个在正码，一个在特码 = 中奖
- 其他 = 不中奖

### 特串

- 选 2 个号码为一组
- 一个在正码，一个在特码 = 中奖
- 两个都在正码 = **不中奖**

## 正码玩法

### 正码1-6特

下注特定位置的正码号码：
- 正1特：猜第一个开出的正码号码
- 正2特：猜第二个开出的正码号码
- ...以此类推
- 号码和顺序都正确 = 中奖

### 正码1-6（单双/大小/色波）

每个正码位置可单独投注：
- 单双
- 大小
- 色波（红/蓝/绿）
- 合数单双

规则与特码相同，49 号为和局（色波为绿波）

## 过关玩法

### 正码过关

在正码里选择多种项目串联成投注组合：
- 赔率 = 所选串赔率的总乘积
- 所有串联项目都中 = 中奖
- 某串为 49 号和局时，该串赔率以 1 计算

---

## 六肖玩法

- 选择 6 个生肖为一组
- 选择【中】：特码在此组合内 = 中奖
- 选择【不中】：特码不在此组合内 = 中奖
- 特码 49 号 = 和局

---

## 备注

- 49 号在大多数玩法中为和局
- 组合玩法中号码需满足最低数量要求
- 生肖号码每年轮换，需根据当年更新
